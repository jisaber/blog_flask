## 运行方式
```
cd ./blog_flask/microblog
flask run --host 192.168.3.14 --port 6002
(http://192.168.3.14:6002/)
```
或者
```
cd ./blog_flask/microblog
flask run
(http://127.0.0.1:5000/)
```
